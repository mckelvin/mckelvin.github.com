virtual void CThostFtdcTraderSpi::OnFrontDisconnected ( int  nReason)
inlinevirtual

当客户端与交易后台通信连接断开时，该方法被调用。当发生这个情况后，API会自动重新连接，客户端可不做处理。

参数
nReason错误原因 0x1001 网络读失败 0x1002 网络写失败 0x2001 接收心跳超时 0x2002 发送心跳失败 0x2003 收到错误报文
virtual void CThostFtdcTraderSpi::OnHeartBeatWarning ( int  nTimeLapse)
inlinevirtual

心跳超时警告。当长时间未收到报文时，该方法被调用。

参数
nTimeLapse距离上次接收报文的时间
